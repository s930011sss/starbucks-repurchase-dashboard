# Starbucks Post-Offer Repurchase Dashboard

This Streamlit app demonstrates a one-page SaaS-style dashboard for post-offer member strategy decisions.

The dashboard is designed around a member-level workflow: after an offer is completed or expires, the interface presents a 72-hour repurchase propensity, readable model indicators, and a recommended marketing action.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files

- `app.py` - Streamlit dashboard prototype.
- `requirements.txt` - Python dependency list.
- `.streamlit/config.toml` - Local Streamlit settings for headless preview.
